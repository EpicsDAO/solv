Update - Solana Validator Mainnet

## Update solv

```bash
$ solv update
$ solv -V
3.3.17
```

## Update Solana Validator Version

```bash
$ solv update -b
```

You can always check the validator status by running the following command:

```bash
$ solv get monitor
```
